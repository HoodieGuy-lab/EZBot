🛡️ Why does Windows show a warning?

When you download EZBot Pro, Windows may show a "Windows protected your PC" message. This is completely normal and does not mean the app is dangerous.

Windows SmartScreen shows this warning for any new app that hasn't been downloaded by thousands of users yet — including totally legitimate software. It has nothing to do with whether an app is safe or not.

EZBot Pro is 100% free, open source, and contains no malware. The source code is publicly available on GitHub for anyone to inspect.

To run the app:
1 Click More info on the blue warning popup
2 Click Run anyway — the app will install normally
3 That's it! You'll never see this warning again for EZBot Pro
